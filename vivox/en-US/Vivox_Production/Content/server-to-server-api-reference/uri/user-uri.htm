User URI structure

A user URI uses the following format:

sip:.issuer.me.@mt1s.vivox.com.

The following list details each part of the user URI structure:

  • sip - indicates a SIP session

  • issuer - the server issuer

  • me - the name of the user account

  • @mt1s.vivox.com - the Vivox server that the user is connecting to

Note: Ensure that the user_uri parameter is a fully qualified identifier.
