XMPP representation - Edit a message in a channel

Edit request

Example edit request:

<iq type='set' id='...' from='...' to='...'> <query xmlns='urn:xmpp:mam:3'> <x xmlns='jabber:x:data' type='submit'> <field var='FORM_TYPE' type='hidden'> <value>urn:xmpp:mam:3</value> </field> <field var='message-id'> <value>123456789</value> </field> <field var='message'> <value>New Message</value> </field> </x> </query> </iq>

Edit response

Example edit response:

<iq from="someRoomSIP@cats.vivox.com" id="..." to="alice@cats.vivox.com" type="result"> <message-edited xmlns="urn:vivox:message-edited" id="..."> <new-message>New Message</new-message> <edit-time>1656521027</edit-time> </message-edited> </iq>

Edited event

Edited event notifications can be propagated back to every participant in the channel.

Example of propagating an edited message event:

<message type=”event” from=”...” to=”...”> <event urn=”urn:vivox:message-edited”> <edit to=”alice@cats.vivox.com” message-id=”...”> <new-message>New Message</new-message> <edit-time>1656521027</edit-time> </edit> </event> </message>
