Add an application API key

When you have enabled the Enable Remote API checkbox, you can automate the process for adding an application API key by calling the REST endpoint located at developer-api.vivox.com/api/v1/applications/remote-api/key/add with a tool such as cURL, Advanced REST Client (ARC), or Postman.

Note: You can have a maximum of two API keys at a time.

The following code displays an example of how to add the “somenewkey” application API key by using cURL:

curl "https://developer-api.vivox.com/api/v1/applications/remote-api/key/add" -X POST -d "{\"remoteApiUsername\":\"remote-api-username-from-org-settings-page\",\"remoteApiPassword\":\"remote-api-password-from-org-settings-page\",\"issuer\":\"issuer-from-app-details-page\",\"key\":\"somenewkey\"}"

A response in the following format indicates that you successfully added the key:

{"statusCode":200,"data":"some-new-key"}
