URI structure for the report API

The following displays an example of how to structure a URI to use with the report API:

  • Domain: mt1s.vivox.com

  • Issuer: blindmelon-AppName-dev

  • Admin user: blindmelon-AppName-dev-Admin

  • Sender: john

  • Target: doe

Based on this information, use the following URI structures:

  • user_uri - "sip:blindmelon-admin@mt1s.vivox.com"

  • sender_uri - "sip:.blindmelon-john.@mt1s.vivox.com"

  • target_uri - "sip:.blindmelon-doe.@mt1s.vivox.com"
