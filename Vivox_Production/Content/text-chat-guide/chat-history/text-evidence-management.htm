Text evidence management

Note: This feature is in Open Beta.

Text Evidence Management is an optional feature that helps moderators evaluate player reports with conversational evidence. If a message was edited, deleted, or redacted by the Chat filter, all versions of the message are stored for review. To enable this feature, reach out to Unity Support or your Vivox representative.

With this server-to-server API, you can request text messages from player chat history to review instances of reported behavior.
