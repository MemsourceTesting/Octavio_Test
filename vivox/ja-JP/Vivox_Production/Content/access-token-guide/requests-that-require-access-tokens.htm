Requests that require access tokens

Each privileged operation requires a token in a specific format. Examples of privileged operations include:

  • Signing in
  • Joining a channel (muted or unmuted)
  • Kicking a user from a channel or from the server
  • Muting or unmuting a user in a channel for all users in the channel
  • Requesting the start of a transcription for an audio session

For more information, see Supported values for the Vivox action claim.
