Evidence report HTTP request

Before requesting an evidence report you will need to acquire your auth_token. The auth_token is generated after a successful user authentication using api/viv_signin.php.

The following shows an example of requesting this auth token:

curl -v "https://mt1s.www.vivox.com/api2/viv_signin.php?userid=userid&pwd=userPassword”

The following shows an example of requesting an evidence report between john and doe:

curl -v "https://mt1s.www.vivox.com/evidence/report? auth_token=blindmelon-admin:1670856702:a1df7a2abebbc707056f9795751d3ec6:10.254.12.34&user_uri=sip:blindmelon-admin@mt1s.vivox.com&sender_uri=sip:.blindmelon-john.@mt1s.vivox.com&target_uri=sip:.blindmelon-doe.@mt1s.vivox.com"
