Chat history

Note: This feature is in Open Beta and is currently only available in the Core SDK.

Chat history allows for the archiving and client-side retrieval of text conversations between users. Users can look up specific conversations or view previously sent messages by all users in a conversation.

Topic Description
Retrieve account-wide chat history How to retrieve account-wide chat history of a user.
Retrieve channel-based chat history How to retrieve channel chat history.
Get last read message information from a given channel How to retrieve message read information.
