Edit and Delete

Note: This feature is in Open Beta and is currently only available in the Core SDK.

With the Edit and Delete functionality you can provide users with the ability to edit or delete previously sent text messages.

Client Interface

The Edit and Delete features have interfaces exposed as listed below:

  • Channel-based message edit.

  • Channel-based message delete.

Topic Description
Channel-based message edit How to edit channel-based messages.
Channel-based message delete How to delete channel-based messages.
XMPP representation - Edit a message in a channel Reference for XMPP representation for editing messages.
XMPP representation - Delete a message in a channel Reference for XMPP representation for deleting messages.
Custom XML Namespaces Reference for custom XML namespaces used by Vivox.
