Prerequisites

The following list details the prerequisites for using the Vivox Server to Server Web API:

  • A Vivox service instance
  • The Vivox domain that you want to control
  • An administrator account and password for that domain
  • Your assigned Vivox API endpoint
  • Your assigned admin user ID and admin password.

Note: The API endpoint, admin user ID, and admin password information is provided when you create an application on the Vivox Developer Portal. If you are using the Unity Dashboard, reach out to your Vivox representative for these details and to enable this feature.
