Fully qualified identifiers

Ensure that the user_uri and chan_uri parameters are fully qualified identifiers.

Note: A fully qualified identifier includes your issuer as a part of the URI.

Fully qualified URIs use the following formats:

  • Users - sip:.issuer.userid.@somedomain.vivox.com
  • Channels - sip:confctl-[g/d/e]-issuer.channelid@somedomain.vivox.com

For more information, see Channel and user identification.
